# Components

Reusable Textual UI components for the trading dashboard.

## Files

| File | Purpose |
|------|---------|
| `__init__.py` | Package exports |
| `ai_panel.py` | AIPanel - AI reasoning and decision log |
| `charts_panel.py` | ChartsPanel - candlestick charts display |
| `goal_panel.py` | GoalPanel - goal progress and AI sizing context |
| `history_panel.py` | HistoryPanel - trade history table |
| `markets_panel.py` | MarketsPanel - market prices, signals, positions |
| `signal_adapter.py` | SignalBrainAdapter - connects signals to UI display |
| `status_bar.py` | StatusBar - connection status, balance, P&L |
