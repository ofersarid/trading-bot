# Styles

CSS theme files for the Textual TUI dashboard.

## Files

| File | Purpose |
|------|---------|
| `theme.css` | Main dark theme with retro terminal aesthetic |

## Hot Reload

CSS changes are hot-reloaded when running in dev mode:

```bash
TEXTUAL_DEV=1 python bot/ui/dashboard.py
# or
./dev.sh <session>
```
