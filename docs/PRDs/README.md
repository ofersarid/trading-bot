# Product Requirements Documents (PRDs)

This folder contains all product and technical documentation for the Crypto AI Trading Bot.

## Documents

| Document | Description | Status |
|----------|-------------|--------|
| [System Architecture](./system_architecture.md) | Overall system design, components, data flow | Draft |
| [Local AI Integration](./local_ai_integration.md) | Ollama integration for local LLM inference | Draft |

## Document Status Legend

- **Draft** - Initial version, subject to major changes
- **Review** - Ready for review, minor changes expected
- **Approved** - Finalized, changes require versioning
- **Deprecated** - No longer current, kept for reference

## Adding New Documents

When adding a new PRD:
1. Create the markdown file in this folder
2. Update this README with the document link
3. Include version, date, and status in the document header

## Template

```markdown
# [Document Title]

> **Version:** 1.0
> **Last Updated:** [Date]
> **Status:** Draft

---

## 1. Overview
...
```
